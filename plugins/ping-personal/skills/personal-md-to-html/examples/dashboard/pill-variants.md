# Pill variants

Default outlined: [[A-TIER]] uses the accent colour on cream background.

Filled variants shown inline:
[[ON-BRAND|good]] green-tint filled pill, [[RISK|warn]] coral-tint filled pill,
[[NEUTRAL|muted]] grey-tint filled pill.

Unknown variant falls back to default: [[BOGUS|unknown]] strips the suffix and
renders as a plain outlined pill — same as [[BOGUS]].

Mixed usage on one line: [[STATUS|good]] indicates all checks passed,
[[BLOCKER|warn]] flags a showstopper, [[WIP|muted]] marks in-progress work.
