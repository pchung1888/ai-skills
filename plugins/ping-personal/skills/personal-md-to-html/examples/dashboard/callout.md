# Callout grid

```callout
- label: CONCENTRATION
  quote: The top 10% of users generate 64% of MRR. The bottom half generates 4%.
  body: Trigger — skewed distribution on a numeric metric. Compute Pareto split.

- label: ANOMALY
  quote: July 14 saw 3.1× the typical signups — and 2× the day-1 churn.
  body: Trigger — any value > 2σ from rolling mean. Cross-reference to second metric.

- label: SURPRISE
  quote: '"annual" users churn at half the rate of "monthly", but downgrade twice as often.'
  body: Trigger — counterintuitive cross-tab. Reserve for the strongest finding only.

- label: SKIP
  quote: If nothing surprising surfaces, write fewer callouts. Don't pad.
  body: '"Three good beats five vague" is in the skill prompt for a reason.'
```
