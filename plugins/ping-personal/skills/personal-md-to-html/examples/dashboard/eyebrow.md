# Body-level eyebrow

```eyebrow
/DASHBOARD SKILL · RESOURCES/CHART-PICKER.MD
```

The block above renders as a small-caps accent breadcrumb — identical styling
to the frontmatter `eyebrow:` field but placed inline within the document body.

---

Regular section prose follows here. The eyebrow sits immediately above this
paragraph and provides path or chapter context.

```eyebrow
PHASE 6 · TASK 7 · BODY-LEVEL EYEBROW
```

A second eyebrow can precede any block — a heading, a paragraph, or a
horizontal rule. It reuses `.eyebrow` from `tokens.css` without redefinition.
