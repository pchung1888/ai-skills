# Walk-mode worked examples

The upstream version of this file contained worked walk renderings against a
specific in-house codebase. They were stripped during the port to keep the
plugin project-agnostic.

To learn the walk-mode shape, read `references/walk.md` -- it documents the
section structure (TL;DR, big-picture diagram, end-to-end trace, knobs,
landmines, where-to-look). The other example files in this folder
(`pr-examples.md`, `qa-examples.md`, `boss-examples.md`) are still intact
and demonstrate the same voice-and-structure conventions applied to PR /
QA / boss modes.

Worked walk examples for your own codebase will appear here once you run
`/personal-htsw walk <file>` a few times and save the renderings.
